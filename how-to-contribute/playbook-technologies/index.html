Technologies used in the playbook

Technologies

The playbook uses these technologies to generate the playbook.

Git and Github

The source files for the Playbook are held in a Git repository hosted at Github.

  • Git tutorial

Markdown

Content is usually written in plain text using Markdown to apply a standardised formatting for things like headings and hyperlinks. Markdown files use a file extension of .md.

  • Markdown tutorial

Eleventy

We use a tool called Eleventy to generate the playbook documentation. If you need more functionality than we’ve described here, take a look at their documentation to find out what’s possible.

If you want to enhance the playbook with more advanced features, you’ll find most of the source under the /lib folder in Github.
